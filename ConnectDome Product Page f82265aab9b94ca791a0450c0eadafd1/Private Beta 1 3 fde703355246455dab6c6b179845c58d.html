Version 1.3b - 11 July 2021

Hey, 👋🏻! Today, we are excited to release 1.3b version of ConnectDome. It’s the fourth update since first private beta release of ConnectDome and includes a lot of exciting changes including new features, remodeled features, bug fixes, and slight UI redesigns.

We also want to share that 10 new users have been added to this 1.3b beta bringing the total to 86 users.

🎉 What's new?

  • 🤖Project AI Recommendation algorithm has been refreshed. Better recommendations await you.
  • 📅 Schedule meetings for 1:1. Powered by Zoom.
    • This feature is in limited preview: One-click 40-minute 1:1 scheduling with your connections and members of your team (if you are either a connection with them or manage them).
    • Note 1: Adding in-build ConnectDome conferencing is on our Roadmap.
    • Note 2: We do not have a time slot conflict check feature yet. However, it's on our roadmap too.
  • 👥 Classic Networking has been replaced by the new "Requests" feature.
    • Send requests to people you already know through the Browse feature.
  • 🔍 Project Search and User Search Introduced.
  • 📝 Applying to Projects is more intuitive now.
  • 👥 Requests. Send requests to users on ConnectDome.
  • 💬 Extended media support in Chat. Preview available in Chat now.
  • ‼️ Report Projects. You can now report a Project if you find it spammy, duplicate, or malicious. Humans carefully review each report.
  • ℹ️ Help buttons added in "Build and Ship" and "Networking" features to help you understand how these features work.

🎯 Up Next

  • Beta 1.4 is up next. It’s going to be a major update as well. We're doing some retooling of our stack with that update and fixing inconsistencies and reiterating the features of 1.3.x releases.
  • We don't have an exact timeline for 1.4b but 1.3.x releases will be regular with 1.3.1 being released in the next 2 days. It will upgrade the limited preview Scheduling system for 1:1 and Team meetings, 1:1 matching algorithm, and a couple more things.

We are working continuously on new features and talking to our users about what our approach should be.
